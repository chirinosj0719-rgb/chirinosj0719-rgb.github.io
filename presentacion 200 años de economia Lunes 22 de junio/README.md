# La exitosa agroexportación y la creciente frontera agrícola

Presentación web interactiva sobre el **Capítulo 35** del libro *200 Años de Economía en el Perú*, para el curso de Historia Económica.

🔗 **Página principal:** [`index.html`](index.html)

## Cómo verla
- Abre `index.html` con doble clic (funciona sin internet).
- Navega con **→ / ← / barra espaciadora**, la rueda del mouse o deslizando.
- Tecla **F** para pantalla completa.

## Publicar en GitHub Pages
1. Sube esta carpeta a un repositorio de GitHub.
2. En el repo: **Settings → Pages**.
3. En *Source* elige la rama `main` y carpeta `/ (root)`.
4. En unos minutos estará en `https://<usuario>.github.io/<repo>/`.

## Estructura
```
index.html        → la presentación (autocontenida)
img/              → fotografías (licencia libre · Creative Commons / dominio público)
GUION DE EXPOSICION.md
```

## Características
- 11 secciones que avanzan punto por punto, estilo editorial kinético (blanco + verde).
- Tractor que avanza y suelta una nube de humo como transición entre slides.
- Fotografías en blanco y negro, tipografía grande y cifras destacadas.

Contenido basado en *200 Años de Economía en el Perú · Capítulo 35*.
